Документация к проету.

- Проект - консольный мессенджер в будущем работающий через сервер.
- Файл account - фактические данные пользователя и функционал для общения. Взаимодействие с сервером происходит через шлюз - manager_request.
- Файл manager_request - это шлюз для работы с api сервером. Никаких данных. Только интерфейс.
- Файл kernel - имитация сервера.
